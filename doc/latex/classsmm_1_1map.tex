\subsubsection*{template$<$typename Key, typename T, size\+\_\+t sz$>$\newline
class smm\+::map$<$ Key, T, sz $>$}

a stack-\/allocation only map datatype 

This class is intended for small (n$<$50) collections, and the algorithms in it reflect that. They do nothing fancy and they perform precisely zero heap allocation.
